Características


¿Qué es Akary?

Akary es un sistema operativo para computadoras de escritorio y portátiles. Está diseñado para funcionar de inmediato y viene completamente equipado con las aplicaciones que la mayoría de las personas necesitas.

Diseño Gráfico

Trabaje en 3D con Blender, dibuje o edite imágenes en Gimp, use Inkscape para gráfico vectoriales.


Productividad

Con la completa suite ofimática de LibreOffice, utilice el procesador de texto, el procesador de textos, haga presentaciones, dibujos, hojas de cálculo o incluso base de datoss. Importe o exporte fácilmente documentos PDFo de Microsoft Office.


Multimedia

Disfruta de tu música, mira televisión y películas, escucha podcasts, Spotify y radio en línea.


Web

Navega por la web, ve YouTube y Netflix con Firefox. Convierte cualquier sitio web en una aplicación de escritorio para acceder al instante.


Juegos

Accede a más de 7800 juegos con Steam. Instala Heroic Game Laucher o Lutris para obtener aún más.
